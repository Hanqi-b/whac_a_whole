# Whac-A-Mole Game - Project Roadmap

Project Overview
A Rust-based Whac-A-Mole game using the Macroquad framework, featuring multiple difficulty levels, diverse enemy types, and AI-assisted development workflow. The project demonstrates proficiency in Rust game development, state management, and sprite-based graphics.

Tech Stack: Rust, Macroquad 0.4, rand 0.8

#### Phase 4: Game Mode 2
- Enhanced enemy variety system:
  - **Normal Moles**: +1 point, single hit
  - **Helmet Moles**: +2 points, requires 3 hits, visual health feedback
  - **Cats**: -5 penalty for misclicks
- Enemy type randomization with weighted spawn rates
- Multi-hit mechanics for armored enemies
- Penalty system for wrong targets
